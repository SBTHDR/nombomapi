# nombomapi

## About Repository

nombomapi is a Food ordering system api using node and express.


## Tech Specification

- Node
- Express
- Mongodb

## Features

- User Registration Endpoint
- User Login Endpoint
- Create, Read, Update, Delete Products Endpoint
- Authorization

## Installation

- `git clone https://github.com/SBTHDR/nombomapi.git`
- `cd nombomapi/`
- `npm install`
- `npm run dev`
